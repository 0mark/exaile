set_loc(self, loc)

source code 

Sets the location. It is always in unicode.

If the value is not unicode, convert it into unicode using some default mapping. This way, when we want to access the file, we decode it back into the ascii and don't worry about botched up characters (ie the value should be exactly identical to the one given)

loc: the location [string]
